# Modelos-de-Clasificación
Dos modelos de clasificación fueron implementados  entrenados y probados con set de datos de Teleco, provisto por IBM y disponible en Kaggle. Los datos son analizados y los modelos son comparados con la finalidad de elegir el que mejor se acople a las necesidades de clasificación de la empresa.

Una vez seleccionado el modelo entonces procedemos a describir posibles aplicaciones y beneficios  que acompañan la implementación del modelo.
